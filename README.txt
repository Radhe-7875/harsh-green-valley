This demo site is generated for Harsh Green Valley Resort (Mahla Ki Dhani, Rajasthan).

Files:
- index.html        (main site)
- styles.css        (styles)
- script.js         (simple form handler)
- sitemap.xml
- README.txt        (this file)

How to deploy for FREE:
1) GitHub Pages:
   - Create a new repository and upload all files.
   - In repository settings, enable GitHub Pages from the main branch.
   - Your site will be available at https://<username>.github.io/<repo-name>/

2) Netlify (recommended):
   - Create a free account at netlify.com
   - Drag & drop the folder into 'Sites' -> 'Create a new site from drag and drop'
   - Netlify will host the site and provide a free HTTPS domain.

Editing content:
- Replace MOBILE NUMBER and EMAIL ID in index.html with real contact details.
- To change images, edit the img src attributes in index.html.

Google Maps:
- The site currently uses an embedded map query for Mahla Ki Dhani, Rajasthan.
- For a precise embed, replace the iframe src with a Google Maps "Share" -> "Embed map" iframe URL.

SEO tips included in the index.html as meta tags and structured data (JSON-LD).
